# airports-search
# Java application that allows you to quickly search for airport data by the text you enter.
# The user launches the application with a parameter that indicates a search for a given column (numbering
starts at 1, not 0).
# After starting the program displays a prompt to enter text in the console.
# After searching, the program displays a list of all matching strings from files on Yandex.Disk, sorted
in the format <Found value of the desired column><Full string> and a suggestion to enter a new search word or a command to exit.
# Search complexity is not optimal, but is O(n) - where n is the number of lines in the file.
